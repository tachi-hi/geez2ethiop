# geez2ethiop

## Installation
```
pip install geez2ethiop
```

## Command Line
First, find the installed script
```
which geez2ethiop
```
Then add the `bin` directory to the `$PATH` if needed.
Then run the following
```
echo አዲስ አበባ | geez2ethiop 
```
